# WEBIMP
Webimp is a web-based image manipulation program.

## Features: 
- Chroma keying 
- Layer masking

## How to start the server: 
MacOS/Linux:

```./mvnw spring-boot:run```

Windows:

```mvnw spring-boot:run```

## Run web application:
 http://localhost:8080
